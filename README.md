# khud 
### clean hud for counter-strike: source
<img src="/khud.png" alt="Screenshot" style="width: 80%">


### installation:
- download the zip and extract it
- open the extracted folder and locate the `custom` folder
- move the entire custom folder into `steamapps/common/Counter-Strike Source/cstrike` directory (replace if prompted)
- *optional: remove khud_skins from the folder if you don't want custom arms / knife*
- *optional: remove khud_sounds from the folder if you don't want csgo weapon sounds*
